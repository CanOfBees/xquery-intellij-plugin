XQuery IntelliJ Plugin

Download

Language support

The XQuery IntelliJ Plugin adds XQuery language support to the IntelliJ IDE.

XQuery

The plugin fully supports XQuery 1.0, 3.0, and 3.1, along with the XPath and XQuery Full Text, XQuery Update Facility 1.0 and 3.0, XQuery Scripting, and EXPath extensions.

Vendor extensions

The plugin fully supports the BaseX and Saxon vendor extensions, along with the MarkLogic 1.0-ml syntax variant. It does not currently support the MarkLogic 0.9-ml syntax variant, or other XQuery-like languages such as JSONiq.

Error recovery

The plugin is capable of recovering from parser errors, and will offer useful hints where possible.
